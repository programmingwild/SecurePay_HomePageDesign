📋 Manual Figma Setup Steps

  1. Create a new Figma file
  2. Set up Color Styles:
    • Create color style for each color above (Background, Text, Accent, etc.)
    • Label them: color/dark/primary, color/light/primary, etc.
  3. Set up Typography:
    • Create text style: "Heading" (Google Sans, 20px, 700)
    • Create text style: "Body" (Google Sans, 13px, 400)
    • Create text style: "Label" (Google Sans, 12px, 600)
    • Create text style: "Amount" (Google Sans, 32px, 800)
  4. Build Button Component:
    • Rectangle: 200×60px, border-radius=14px
    • Apply primary color, add text
    • Create "Default" and "Pressed" (0.95 scale) variants
  5. Build Card Component:
    • Rectangle with rgba(255,255,255,0.08) fill
    • Stroke: 1px rgba(255,255,255,0.1)
    • Effects: Blur 10px
    • Border-radius: 16px
  6. Build Quick Actions Grid:
    • 8 tiles, 80×80px each, 10px gap
    • Each tile: colored background, icon, label below
    • Colors from "Quick Actions Colors" above
  7. Create Phone Frame:
    • Rectangle: 1080×2400px
    • Stroke: 1.3px #25345B
    • Border-radius: 32px
    • Shadow: 0 16px 30px rgba(0,0,0,0.66)
