# SecurePay Homepage -> Figma

This workspace cannot export a native `.fig` binary directly, but this file gives you a Figma-importable design source.

## Import steps
1. Open Figma.
2. Create a new design file.
3. Drag and drop `design/securepay_homepage_figma_import.svg` onto the canvas.
4. Figma will import it as editable vector/text layers.

## Notes
- Frame size: 1440 x 2600
- Typeface used in design spec: Google Sans
- This SVG mirrors the current homepage structure and visual direction (glassmorphism + immersive fintech style).
